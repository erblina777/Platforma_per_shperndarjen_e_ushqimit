# Food Delivery Platform

## Përshkrimi:
Ky projekt është një platformë web për porositjen e ushqimit online.
Ai lidh klientët, restorantet dhe shoferët për të menaxhuar porositë dhe dërgesat.

## Funksionalitetet:
Regjistrimi dhe kyçja e përdoruesve
Menaxhimi i restoranteve dhe menusë
Krijimi i porosive
Menaxhimi i dërgesave
Vlerësime dhe komente
